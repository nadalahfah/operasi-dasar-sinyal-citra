## 1. Deskripsi Singkat Project
Project ini berfokus pada implementasi dan analisis operasi aritmatika dasar pada pemrosesan sinyal 1-Dimensi (audio) dan citra 2-Dimensi (gambar). Eksperimen meliputi proses penjumlahan, penggeseran posisi, amplifikasi nilai, serta pengujian karakteristik kelinieran sistem menggunakan prinsip superposisi (uji homogenitas dan additivitas).
